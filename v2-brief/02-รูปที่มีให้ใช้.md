# รูปที่เจนเสร็จแล้วและใช้ได้จริง

รูปทั้งหมดอยู่ที่ `เกมที่3/app/public/assets/` (คัดลอกไปใช้ได้เลย)
ทุกใบเป็นภาพที่เจนมาเพื่อเกมนี้โดยเฉพาะ **สไตล์เดียวกันทั้งชุด**: การ์ตูนแบน ขอบดำหนา สีสดจัด
โทนหลัก น้ำเงิน `#2b4bd8` · ม่วง `#6a2fd0` · ชมพูบานเย็น `#e82f9c` · เหลืองทอง `#ffd93d` · ขอบดำ `#10121f`

**"โปร่งใส"** = มี alpha ใช้วางทับอะไรก็ได้ · **"ทึบเต็มเฟรม"** = ภาพพื้นหลังเต็มใบ ไม่มี alpha

**เรื่องที่ต้องรู้เกี่ยวกับฉากปะทะ:** ในภาพมีที่ว่างเจาะไว้ให้แล้ว (วัดเป็น % ของภาพ):

| ช่องว่างในภาพ | ตำแหน่ง |
|---|---|
| กรอบสี่เหลี่ยมฝั่งซ้าย (ไว้วางรูปคน) | แนวนอน 10–34% · แนวตั้ง 26–65% |
| กรอบสี่เหลี่ยมฝั่งขวา (ไว้วางรูปคน) | แนวนอน 66–91% · แนวตั้ง 26–65% |
| วงกลมกลางจอ (ไว้วางตราปะทะ) | จุดกึ่งกลาง 50%, 46% · เส้นผ่านศูนย์กลาง ~19% |
| แถบยาวด้านล่าง (ไว้ใส่ข้อความ) | แนวนอน 4–96% · แนวตั้ง 83–96% |

ซีกซ้ายกับซีกขวาถูกผ่าจากใบเต็มด้วยมาสก์ที่ complement กันพอดี → **วางซ้อนกันแล้วได้ภาพเต็มกลับมาเป๊ะ**

## กฎการใช้รูป

1. **ห้ามวาดภาพขึ้นมาเองด้วยโค้ด** (SVG art / canvas drawing / emoji แทนภาพ) — ใช้ไฟล์ที่มีเท่านั้น
2. **ตัวอักษรไทยในภาพห้ามเจนใหม่** — AI เขียนไทยผิดเสมอ ข้อความทั้งหมดต้องเรนเดอร์เป็นข้อความจริงในหน้าเว็บ
3. ถ้าต้องการภาพที่ยังไม่มี → **ใส่ที่ว่างหรือไอคอนชั่วคราวไว้ก่อน** แล้วเขียนรายการไว้ว่าอยากได้ภาพอะไรเพิ่ม เดี๋ยวจะเจนให้ทีหลัง
4. รูปผู้เล่นจริงเป็นรูปสี่เหลี่ยมจัตุรัสที่ผู้ใช้ถ่าย/วางลิงก์เอง (ไม่ใช่ asset ในชุดนี้) — คนที่ไม่มีรูปใช้ `avatar-placeholder`
