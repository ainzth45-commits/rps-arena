## 🔧 คำสั่งที่ใช้บ่อย

```bash
cd เกมที่3/app
npm run test              # เทส 110 ตัว
npm run build             # tsc + vite
node verify-flow.mjs      # เดิน 17 จอ วัด overflow (ต้องเสิร์ฟ dist ก่อน)
node verify-offround.mjs  # เทสดวลนอกรอบ

# เสิร์ฟ dist ทดสอบ (Playwright ต้องการ):
ln -sfn dist rps-arena && python3 -m http.server 8902
# แล้ว verify ชี้ localhost:8902/rps-arena/

# sync asset จาก Codex เข้าเกม:
cp theme-samples/assets-batch/XXX.webp app/public/assets/{icons,chars,scenes}/
# แล้วเพิ่มใน app/src/data/assets.ts + bump V (?v=N ล้าง cache)
```

**deploy:** push main → GitHub Actions → `ainzth45-commits.github.io/rps-arena/` อัตโนมัติ
