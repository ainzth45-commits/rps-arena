# Theme Samples V2

ธีมบอร์ดรอบใหม่สำหรับเกม "เป่า ยิ้ง ฉุบ! อารีน่า!" ปรับทิศทางจากภาพกึ่งสมจริงเป็นการ์ตูนแบนกวนๆ แบบเกมมือถือ casual

ข้อกำหนดหลักของรอบนี้:

- ไม่มีตัวอักษรในภาพ
- ด้านบนเป็นแผ่นป้ายโลโก้เปล่า สำหรับวางชื่อภาษาไทยด้วยฟอนต์จริงภายหลัง
- ไอคอนมูฟทั้ง 3 ชิ้นเป็นท่ามือเป่ายิ้งฉุบเท่านั้น: กำหมัด / ชูสองนิ้ว / แบมือ
- สไตล์หลักคือสีแบนสด ขอบดำหนา รูปทรงอ่านง่าย หน้าตาตลกเกินจริง
- ตัวละครหลักคือมาสคอตแมวส้มจากเกมที่ 1 ในโหมดนักสู้ ปะทะกับพนักงานหญิงออฟฟิศ

## แบบ 1 — Hot Radial Arena

- ไฟล์: `theme-v2-1.png`
- โทนสี: ส้ม แดง เหลือง และประกายขาว
- ภาษาภาพ: ใกล้ reference แมว vs หมา มากที่สุด ใช้วงรัศมีพุ่ง ฉากสังเวียนเรียบ และเอฟเฟกต์ปะทะกลางจอ
- จุดขาย: อ่านง่ายที่สุด สนุกที่สุด และมีความ casual mobile game ชัดมาก เหมาะถ้าต้องการให้เจ้าของเกมเห็นทันทีว่า "นี่ไม่ใช่ภาพสวยจริงจัง แต่เป็นเกมฮาๆ"
- ความเสี่ยง: โทนร้อนอาจคล้ายเกม casual ทั่วไป และอาจต้องคุม UI ภายหลังไม่ให้สีแดง/ส้มแย่งความสำคัญกับปุ่มหลัก

## แบบ 2 — Electric Clash Arena

- ไฟล์: `theme-v2-2.png`
- โทนสี: ฟ้า น้ำเงิน ม่วง ชมพู และฟ้าผ่าขาว
- ภาษาภาพ: ใกล้ reference clash มากขึ้น ใช้พลังปะทะกลางจอแบบแบ่งฝั่ง สีเย็นตัดชมพู และฉากเชียร์ในสังเวียน
- จุดขาย: ให้ความรู้สึกแข่งขัน ดวล และ "มันส์" มากกว่าแบบ 1 เหมาะกับหน้า VS, streak, ranking และจังหวะเปิดผลแพ้ชนะ
- ความเสี่ยง: ความนีออนอาจทำให้ mood ห่างจากความ office เล็กน้อย และต้องระวังไม่ให้กลายเป็นอาร์เคดจริงจังเกินไป

## แบบ 3 — Office Battle League

- ไฟล์: `theme-v2-3.png`
- โทนสี: มิ้นต์ เขียวอมฟ้า คอรัล เหลือง และม่วงเล็กน้อย
- ภาษาภาพ: สังเวียนออฟฟิศแฟนตาซี โต๊ะประชุมกลายเป็นสนามแข่ง เก้าอี้และสปอตไลต์เป็นฉากหลัง พร้อม confetti/post-it เปล่าๆ ไม่มีตัวอักษร
- จุดขาย: ผูกกับบริบทพนักงานออฟฟิศและยอดขายชัดที่สุด แต่ยังคงความแบน กวน สดใส และอ่านง่ายแบบเกมมือถือ
- ความเสี่ยง: อาจต้องคุม asset รอบต่อไปไม่ให้พร็อพออฟฟิศรกเกินไป โดยเฉพาะเมื่อต้องทำ UI จริงบน iPad

## Recommendation

ฉันแนะนำแบบ 3 — Office Battle League เป็นตัวเลือกหลัก เพราะตอบโจทย์ "เกมที่คนออฟฟิศเล่นกันจริง" ได้ตรงที่สุด และช่วยให้เกมนี้ต่างจากเกมเป่ายิ้งฉุบทั่วไปทันที เห็นแล้วเข้าใจว่าเป็นจักรวาลทีมเดียวกับเกมที่ 1 แต่เปลี่ยนจากนักสืบมาเป็นสังเวียนยอดขาย

ถ้าเจ้าของเกมอยากได้ความฮาแบบ reference แมว vs หมา เป๊ะที่สุด ให้เลือกแบบ 1 ส่วนแบบ 2 เหมาะถ้าต้องการดันหน้า VS ให้ลุ้นและมีพลังปะทะมากที่สุด
