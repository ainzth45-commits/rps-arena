## แบ่งงานกันยังไง

| ใคร | รับผิดชอบ |
|---|---|
| **ฟรายเดย์** | โค้ดทั้งหมด · โดเมน/สเตต/หน้าจอ · เทส · verify · deploy · เอกสาร |
| **Codex** | **งานภาพทั้งหมด** (ธีม โลโก้ ไอคอน ฉาก) + **รีวิวโค้ด** |

---

## เฟส

### 🎨 P0 — เคาะธีม (กำลังทำ · บล็อกงานอาร์ตทั้งหมด)

Codex เจนธีมบอร์ด 3 แนว → เจ้านาย approve → เขียน `02-art-direction.md`

**ทำไมต้องมาก่อน:** ธีมกำหนดสี ฟอนต์ อารมณ์ และสไตล์ asset ทุกชิ้น ถ้าเลือกทีหลังต้องรื้อ CSS กับภาพใหม่หมด
**ไม่บล็อก:** งานโดเมน/ตรรกะเกม (P1–P2) ทำคู่ขนานได้เลย เพราะไม่เกี่ยวกับภาพ

---

### ⚙️ P1 — โครงโปรเจกต์ + เครื่องยนต์เกม

ตั้ง Vite + React + TS + PWA + git + GitHub Actions deploy
เขียน **โดเมนล้วน** (ฟังก์ชันบริสุทธิ์ เทสง่าย ไม่มี UI):
- ตัดสินแพ้ชนะเป่ายิ้งฉุบ
- คำนวณคะแนน + ตัวคูณสตรีค + พื้นที่ 0
- ตัวชี้ชุดมูฟ (เดิน/รีเซต)
- จัดอันดับ 4 ชั้น + อันดับร่วมแบบกีฬา
- เรตมูฟท็อป 3

**เสร็จเมื่อ:** เทสโดเมนผ่านครบ รวม edge case ทั้ง 13 ข้อใน spec §16

---

### 💾 P2 — สเตต + ผู้เล่น

- โครงสร้างเซฟ + localStorage + migration + กันเซฟพัง (แถบเตือนแบบเกมที่ 1)
- ลงทะเบียนผู้เล่น (รหัส 4 ตัว · รูป 3 ทาง) — **ยกโค้ดจากเกมที่ 1 มาปรับ**
- หน้าตั้งค่าแอดมิน

**เสร็จเมื่อ:** เพิ่ม/แก้/ลบผู้เล่นได้จริง ปิดแอปเปิดใหม่ข้อมูลอยู่ครบ

---

### 🥊 P3 — วงจรหลักของเกม (ก้อนใหญ่ที่สุด)

หน้าจอ 3–11: เลือกผู้เล่น → 📬 ระหว่างที่คุณไม่อยู่ → เมนูรอบ → ปรับชุดมูฟ → เลือกผู้ท้าชิง → VS → เลือกมูฟ (30 วิ) → ฉากดวล → ผล

**เสร็จเมื่อ:** เล่นครบวงจรได้จริงตั้งแต่ต้นจนจบรอบ คะแนนขึ้นถูกต้อง ตัวชี้เดินถูก

---

### 🏆 P4 — อันดับ + ประวัติ

- ตาราง ranking + กดดูเรตมูฟท็อป 3
- ประวัติของฉัน (แยกแท็บผู้เล่น/ผู้ท้าชิง)
- ป้ายเตือน "โดนไล่เก็บ"

---

### 🎪 P5 — ดวลนอกรอบ

ผลัดกันเลือกมูฟ + จอปิดทับตอนส่งเครื่อง + บันทึก 3 ทาง

---

### 📅 P6 — ซีซั่น

จบซีซั่น → ประกาศแชมป์ → บันทึก record ถาวร → เปิดซีซั่นใหม่ (ล้างชุดมูฟ)

---

### 🖼️ P7 — ใส่งานอาร์ตจริง + เสียง

Codex ส่ง asset ตามธีมที่ approve → ฟรายเดย์ใส่ลงเกม + ทำ PWA icon + ใส่ซาวด์

**ป้ายข้อความไทยในภาพ ฟรายเดย์ประกอบเองด้วย ImageMagick + ฟอนต์ Kanit** (บทเรียนเกมที่ 1 — AI เจนตัวอักษรไทยเพี้ยนทุกครั้ง)

---

### 📖 P8 — สอนเล่น

ฉากสอนเล่นแบบเกมที่ 1 — เน้นกลไกที่คนงงแน่ๆ: ตัวชี้ชุดมูฟ · ทำไมต้องจ่ายเหรียญเปลี่ยนเซต · ภาษีของแชมป์

---

### ✅ P9 — ขัดเงา + ส่งมอบ

- ไล่ทุกหน้าให้ฟิตจอ iPad ไม่ต้องเลื่อน (วัดด้วย Playwright 1180×820)
- **Codex รีวิวโค้ดทั้งหมด** → ฟรายเดย์แก้
- เทสครบ + deploy GitHub Pages + เจ้านายเทสบน iPad จริง

---

## ลำดับการพึ่งพา

```
P0 ธีม ─────────────────────────► P7 อาร์ต ──► P8 สอนเล่น ──► P9 ส่งมอบ
                                      ▲                          ▲
P1 เครื่องยนต์ ─► P2 สเตต ─► P3 วงจรหลัก ─► P4 อันดับ ─► P5 นอกรอบ ─► P6 ซีซั่น
```

**P0 กับ P1–P2 เดินคู่ขนานได้** — ตอนนี้ Codex ทำ P0 อยู่ ฟรายเดย์เริ่ม P1 ได้เลย

---

## เกณฑ์ "เสร็จ" ของทั้งโปรเจกต์

1. เล่นได้ครบวงจรบน iPad จริง ไม่มีหน้าไหนต้องเลื่อนจอ
2. เทสอัตโนมัติผ่านทั้งหมด ครอบคลุม edge case ใน spec §16
3. Codex รีวิวโค้ดแล้วไม่มีประเด็นค้าง
4. deploy ขึ้น GitHub Pages เล่นได้จากลิงก์จริง
5. **เจ้านายเล่นแล้วบอกว่าผ่าน**
