## 🎯 แก่นเกม

เกมเป่ายิ้งฉุบสะสมคะแนนแบบ **อารีน่า/แรงกิ้ง** เล่นยาวข้ามวันจนจบ **season** (เจ้านายเป็นคนกดจบเอง)
จบ season → บันทึกอันดับไว้โชว์ → แจกรางวัลจริงตามอันดับ

**จุดกระตุ้นยอดขาย:** เล่น 1 ครั้ง = จ่าย **3 เหรียญ** · เหรียญได้จากยอดขาย/กิจกรรมจริง
พนักงานเดินไปหาซุปเพื่อจ่ายเหรียญ แล้วขอเล่น 1 ครั้ง

## 🎮 กติกาที่เจ้านายวางไว้

### 1. ลงทะเบียน + ตั้ง move set

ผู้เล่นทุกคนลงทะเบียนก่อน แล้วตั้ง **ชุดมูฟ** ของตัวเอง 3 ช่อง:

```
เป่า1 : ค้อน
เป่า2 : กรรไกร
เป่า3 : กรรไกร      ← ซ้ำกันได้
```

### 2. จ่ายเหรียญ → ได้ 1 รอบ

1 รอบทำได้ **2 อย่าง** จะทำอันไหนก่อนก็ได้:

| ทำได้ | รายละเอียด |
|---|---|
| **ดวล** | ท้าดวลกับคนอื่น 1 ครั้ง |
| **ปรับแต่งเซต** | แก้ เป่า1/เป่า2/เป่า3 · เปลี่ยนกี่ช่องก็ได้ · **ทำได้ 1 ครั้งต่อรอบ** · ไม่อยากเปลี่ยนก็ข้ามได้ |

### 3. เลือกผู้ท้าชิง

- เลือกเองว่าจะดวลกับใคร — **เลือกแล้วเปลี่ยนไม่ได้**
- หรือกด **สุ่ม** ถ้าไม่รู้จะดวลกับใคร → **ได้คะแนนเยอะกว่า เสียคะแนนน้อยกว่า**

### 4. หน้าเตรียมตัว (30 วิ)

ผู้เล่นเลือกจะออกอะไร: ค้อน / กรรไกร / กระดาษ → กดยืนยัน
**เลือกไม่ทัน = ระบบสุ่มให้**

### 5. หน้าเป่ายิ้งฉุบ ← จังหวะลุ้นสุดของเกม

ผู้ท้าชิง**ไม่ต้องอยู่** — ระบบออก move แทนตามเซตที่เจ้าตัวตั้งไว้

**กลไกหัวใจ: ตัวชี้เดินหน้าทีละช่อง**

- ครั้งแรกที่ถูกท้า → ออก **เป่า1** แล้วจำไว้
- ครั้งถัดไปที่ถูกท้า → **เป่า2** → **เป่า3** → วนกลับ **เป่า1** ไปเรื่อยๆ
- **ไม่สนว่าใครเป็นคนท้า** — ตัวชี้ผูกกับตัวผู้ท้าชิงคนนั้น
- **รีเซตกลับไป เป่า1 เมื่อเจ้าตัวจ่ายเหรียญมาเปลี่ยนเซตของตัวเอง**

> นี่คือเกมจิตวิทยาของเกมนี้ — จำได้ว่าใครออกมูฟอะไรไปแล้ว = เดามูฟต่อไปได้
> เจ้าตัวก็ต้องคอยจ่ายเหรียญมาสับขาหลอก

### 6. คะแนน

**ผู้เล่น (คนจ่ายเหรียญ) — เลือกคู่ต่อสู้เอง**

| ผล | คะแนน |
|---|---|
| ชนะ | +4 |
| เสมอ | +1 |
| แพ้ | −3 |

**ผู้เล่น — กดสุ่มคู่ต่อสู้**

| ผล | คะแนน |
|---|---|
| ชนะ | +5 |
| เสมอ | +1 |
| แพ้ | −2 |

**ผู้ท้าชิง (ฝ่ายถูกเลือก · ระบบเล่นแทน)**

| ผล | คะแนน |
|---|---|
| ชนะ | +3 |
| เสมอ | +1 |
| แพ้ | −2 |

**โบนัสชนะต่อเนื่อง**

- นับ**เฉพาะตอนเป็นผู้เล่น**เท่านั้น (แพ้/ชนะตอนเป็นผู้ท้าชิงไม่นับ ไม่ตัดสตรีค)
- ชนะติดครั้งที่ 2 = **110%** · ครั้งที่ 3 = **120%** · เพิ่มทีละ 10% สะสมไปเรื่อยๆ
- **ไม่ชนะ (เสมอหรือแพ้) = เรทกลับมา 100%**

### 7. หลังจบการดวล

- บอกผลแพ้/ชนะ + คะแนนที่ได้รอบนั้น
- **ดูอันดับ ranking ได้ทันที**
- มีสถิติแยก: ผลแพ้ชนะ**ตอนเป็นผู้เล่น** vs ผลแพ้ชนะ**ตอนเป็นผู้ท้าชิง**

### 8. Ranking

- จัดตามคะแนนรวม
- คะแนนมีทศนิยม (จะกี่ตำแหน่งค่อยดูตามเรท)

## ✅ กติกาที่เคาะเพิ่มแล้ว (รอบ brainstorm 2026-07-20)

### ก. ป้องกันการ "ฟาร์มผู้ท้าชิง"

ปัญหา: ดวลคนเดิม 3 ครั้งก็อ่านเซตออกครบ แล้วไล่เก็บ −2 ได้เรื่อยๆ โดยเจ้าตัวไม่รู้ตัว

**เลือกทางแก้: ให้ผู้ท้าชิงรู้ตัว (ทางเดียว)** — ไม่ใส่คูลดาวน์ ไม่ใส่ตัวชี้สุ่ม ไม่ใส่เพดานสตรีค

- ❌ **คูลดาวน์ห้ามท้าคนเดิมซ้ำ** — พับไว้ ผู้เล่น 10+ คน และปุ่มสุ่มก็ล่อให้กระจายเป้าอยู่แล้ว
- ❌ **ตัวชี้มีโอกาสหลุดมูฟ** — ตัดทิ้ง **เหตุผลสำคัญ: ถ้าเดาเซตไม่ได้ชัวร์ คนก็ไม่มีแรงจูงใจจ่ายเหรียญมาเปลี่ยนเซต = ฆ่าท่อยอดขายของเกม**
- ❌ **เพดานสตรีค** — ไม่ใส่ ความต่อเนื่องคือความท้าทาย ปล่อยให้คูณไปเรื่อยๆ

### ข. จอ "📬 ระหว่างที่คุณไม่อยู่" + ประวัติ

- **บอกหมดเปลือก**: ใครท้า + **มูฟ**ที่ระบบออกแทนเรา + ผลรายครั้ง + คะแนน + ป้ายเตือนเมื่อมีคนไล่เก็บ
- **ดูย้อนหลังได้ตลอดทั้งซีซั่น**
- **เห็นได้เฉพาะประวัติของตัวเองเท่านั้น** — ห้ามดูของคนอื่น (ถ้าเปิดให้ดูข้ามคน = เดินไปอ่านเซตทุกคนฟรี ท่อยอดขายรั่ว)
- **ประตูเข้าเกม = ซุปเป็นคนกด** — ผู้เล่นจ่ายเหรียญกับซุป → ซุปกดเข้าหน้าของผู้เล่นคนนั้น
  → ไม่ต้องมีระบบรหัส/ล็อกอิน ซุปคือด่านกันเอง
- แรงกิ้ง/คะแนนรวม = สาธารณะ ทุกคนดูได้

### ค. การรีเซตตัวชี้

- ตัวชี้กลับไป **เป่า1** เมื่อผู้เล่น **กดยืนยันในหน้าปรับแต่ง move set เท่านั้น**
- **เข้าไปดูเฉยๆ แล้วถอยออก = ไม่รีเซต** (ทบทวนเซตตัวเองได้โดยไม่เสียของ)
- **ยืนยันโดยไม่เปลี่ยนอะไรเลย = รีเซตด้วย** → จ่าย 3 เหรียญ "ซื้อการสับขาหลอก" ได้
  (คนอื่นนับอยู่ว่าเราต้องออก เป่า3 แล้ว แต่จริงๆ ย้อนกลับไป เป่า1)
- ยืนยันปรับแต่งได้ **1 ครั้งต่อรอบ**

### ง. คะแนนรวม + การจัดอันดับ

- **ทุกคนเริ่มที่ 30 แต้ม**
- **พื้นที่ 0 — ไม่ติดลบ** · ถ้าอยู่ที่ 0 แล้วแพ้ = ไม่เสียคะแนน **แต่บันทึกสถิติไว้** (ไปเจ็บที่เกณฑ์รองแทน)
- **ทศนิยม 1 ตำแหน่ง** — เรทปัจจุบันลงตัวพอดีเป๊ะเสมอ ไม่ต้องปัดเลย (4×1.1=4.4 · 5×1.9=9.5)

**ลำดับการจัดอันดับ (ไล่ทีละชั้น):**

| ชั้น | เกณฑ์ | ทิศทาง |
|---|---|---|
| 1 | **คะแนนหลัก** | มากกว่าชนะ |
| 2 | **คะแนนรอง** (คอลัมน์แยก ไม่บวกเข้าคะแนนหลัก · จำนวนเต็ม) | มากกว่าชนะ |
| 3 | **จำนวนชนะ − จำนวนแพ้** (เสมอไม่นับ) | มากกว่าชนะ |
| 4 | **จำนวนครั้งที่ลงเป็นผู้เล่น** | มากกว่าชนะ (ยิ่งขยันท้าดวลยิ่งได้เปรียบ = จ่ายเหรียญเยอะ) |
| — | เท่ากันหมดทุกชั้น → **ปล่อยให้อันดับเท่ากัน** แล้วค่อยไปตัดสินกันเอง (ใช้โหมดดวลนอกรอบได้พอดี) | |

> **ทำไมคะแนนรองต้องแยกคอลัมน์:** คะแนนหลักมีทศนิยมอยู่แล้วจากตัวคูณสตรีค (4.4 · 5.5) ถ้าเอาคะแนนรองไปบวกรวม มันจะกลายเป็นหน่วยเดียวกันแล้วดันคนแซงกันจริง — ผิดเจตนาที่เจ้านายต้องการ พอแยกคอลัมน์แล้วมันจะไม่มีทางเบียดอันดับใครเลย และไม่ต้องทำเป็นทศนิยมให้ยุ่ง
>
> **ไม่ใช้ตัวคูณ ×2 แล้ว** — พอ "จำนวนครั้งที่เป็นผู้เล่น" แยกเป็นชั้นล่างแทนการเอามาบวก โทษการแพ้ก็ไม่ถูกหักล้าง ไม่ต้องถ่วงน้ำหนักอะไรอีก
>
> **ตัดออก:** สตรีคสูงสุด · ใครถึงคะแนนก่อน (เจ้านายเคาะว่าเอาแค่ 2 ชั้นสถิติพอ)
>
### ฉ. ⚔️ โหมด "ดวลนอกรอบ" (เจ้านายเพิ่ม)

โหมดพิเศษไว้ใช้ตอนจัดกิจกรรม — **ดวลกันจริง ทั้งสองฝ่ายอยู่ตรงนั้น ไม่มีระบบเล่นแทน**

- **ผลัดกันเลือกมูฟ** (ส่ง iPad ให้กัน คนละที) → กดดูผล
- จบแล้ว **มีให้เลือกว่าจะบันทึกคะแนนไหม** → ไม่บันทึก = เล่นสนุกเฉยๆ ไม่กระทบอันดับ
- ครอบคลุมเคส "ดวลตัดเชือก" ตอนคะแนนเสมอกันปิดซีซั่นได้ด้วย

**เรทเบา (ใช้กับทั้งสองฝ่าย):** ชนะ **+2** · เสมอ **+1** · แพ้ **−1**

**ตอนจบมีให้เลือก 3 ทาง:**

| เลือก | ผล |
|---|---|
| **บันทึกเป็นคะแนนหลัก** | เข้าคะแนนหลัก (เรทเบา) + **นับเข้าสถิติแพ้/ชนะ** (มีผลกับชั้น 3 ของอันดับ) |
| **บันทึกเป็นคะแนนรอง** | เข้าคอลัมน์คะแนนรองอย่างเดียว (เรทเบา จำนวนเต็ม) ไม่แตะคะแนนหลัก ไม่แตะสถิติ |
| **ไม่บันทึก** | เล่นสนุกเฉยๆ ไม่กระทบอะไรเลย |

**กติกาเพิ่ม:**
- **ไม่นับสตรีคชนะต่อเนื่อง** ไม่ว่าจะบันทึกแบบไหน (สตรีคเป็นของเกมหลักเท่านั้น)
- **ไม่ขยับตัวชี้ชุดมูฟของทั้งคู่** — เพราะไม่มีใครถูกระบบเล่นแทน ถ้าขยับ คนที่ถูกลากมาดวลโชว์จะเสียตำแหน่งตัวชี้ฟรี

**ยังต้องเคาะ:**
- ต้องจ่ายเหรียญไหม หรือซุปเปิดได้อิสระเพราะเป็นกิจกรรม
- ถ้าบันทึกเป็นคะแนนหลัก นับเป็น "จำนวนครั้งที่ลงเป็นผู้เล่น" (ชั้น 4) ของทั้งคู่ไหม

### ช. จบซีซั่น / ขึ้นซีซั่นใหม่

- ซุปกดจบซีซั่น → หน้าประกาศแชมป์ → **บันทึก record ถาวร ดูย้อนหลังได้ทุกซีซั่น**
- ขึ้น SS ใหม่: คะแนนหลักกลับไป **30** ทุกคน · คะแนนรองล้าง · สถิติล้าง
- **ล้างชุดมูฟทั้งหมด ทุกคนต้องมาตั้งใหม่** (เจ้านายเคาะ — จงใจสร้าง "ช่วงเวลา relax" ให้ทีมได้มารวมตัวตั้งชุดมูฟกันใหม่เป็นอีเวนต์เปิดซีซั่น)
- ตัวชี้รีเซตกลับ เป่า1 โดยปริยาย (เพราะชุดมูฟถูกล้าง)

### ซ.1 เปิดซีซั่น — "ลงสังเวียน"

- **ตั้งชุดมูฟครั้งแรกของซีซั่น = ฟรี ไม่ต้องจ่ายเหรียญ** (เป็นการลงทะเบียนเข้าแข่ง ไม่ใช่การสับขาหลอก)
- **คนที่ยังไม่ตั้งชุดมูฟของซีซั่นนี้ = ไม่โผล่ในรายการเลือกผู้ท้าชิง และไม่มีทางโดนสุ่มโดน**
  ตั้งเสร็จเมื่อไหร่ถึง "ลงสังเวียน" ให้คนอื่นท้าได้
  → กันคนโดนลากไปดวลทั้งที่ยังไม่พร้อม + เป็นแรงกดดันทางสังคมให้รีบมาตั้ง

### ซ.2 หน้าเลือกผู้ท้าชิง

- **ไม่เห็นตัวเองในรายการเลย** (ไม่ใช่แค่กันไม่ให้กด — ไม่แสดงเลย) · ปุ่มสุ่มก็ไม่มีทางโดนตัวเอง
- **แสดง "อันดับ" ของแต่ละคน แต่ไม่แสดงคะแนน** — คะแนนไปดูที่หน้า ranking หรือหน้าผลหลังจบดวล
- เลือกแล้ว/สุ่มแล้ว **เปลี่ยนไม่ได้**

### ญ. 👑 ภาษีของแชมป์ — เปิดเรตมูฟของท็อป 3

**"การเป็นท็อปคือการถูกถอดเสื้อเกราะ หลีกเลี่ยงไม่ได้"** (คำเจ้านาย)

ใครก็กดดูได้จากหน้า ranking — เอาไว้วิเคราะห์ก่อนตัดสินใจท้าดวลกับอันดับบนๆ

| อันดับ | เปิดอะไร |
|---|---|
| **1** | เรตทั้ง **3 มูฟ** (ค้อน / กรรไกร / กระดาษ) |
| **2 และ 3** | เรต**เฉพาะมูฟที่ออกบ่อยที่สุด** มูฟเดียว |

- คิดจาก **จำนวนครั้งทั้งหมดที่เล่นในซีซั่น รวมตอนเป็นผู้ท้าชิงด้วย**
- **โชว์จำนวนครั้งกำกับเสมอ** เช่น "✊ ค้อน 45% · จาก 20 ครั้ง" — กันคนอ่านผิด (เล่น 2 ครั้งออกค้อนหมด = 100% หลอกตา)
- อันดับ 2/3 ถ้ามูฟสูงสุด**เท่ากันหลายอัน → โชว์ทั้งหมดตามจริง**
- **อันดับร่วมใช้แบบกีฬา** — อันดับ 1 ร่วม 2 คน → โชว์ครบ 3 มูฟทั้งคู่ · คนถัดไปเป็นอันดับ 3 (ไม่มีอันดับ 2) · ยึดเลขอันดับที่โชว์บนกระดานเสมอ
- **ดวลนอกรอบนับเข้าเรตนี้เฉพาะตอนบันทึกเป็นคะแนนหลัก**
- **อัปเดตสดทุกครั้งที่มีการดวล** · ทุกคนดูได้ตลอดเวลา

> 🎲 **ผลข้างเคียงที่รู้ไว้:** คนใกล้ขึ้นท็อป 3 อาจลังเล เพราะขึ้นไปแล้วโดนเปิดไพ่
> ถือเป็นเรื่องดี — สร้างดราม่าช่วงชิงโพเดียมท้ายซีซั่น · ถ้าเจอในสนามจริงว่าคนหลบไม่ยอมขึ้นท็อป ค่อยปรับ

### ฌ. คำศัพท์ในเกม

- เรียกว่า **"มูฟ"** ไม่ใช่ "ไม้" (เจ้านายสั่ง — "ไม้" ไม่เท่)
- ชุด 3 ช่องเรียกว่า **"ชุดมูฟ" / "move set"**

## 🧠 บทเรียนจากเกมที่ 1 ที่ควรยกมาใช้

- **ทุก flow ต้องมี "ช็อตลุ้น / ช็อตรอ / ช็อตดูผล"** ไม่ใช่กดแล้วเด้งผลเลย ← เกมนี้ยิ่งสำคัญ

- รูป role/dock/icon = พื้นโปร่งใส · รูปฉากจบ = พื้นหลังเต็มเฟรม
