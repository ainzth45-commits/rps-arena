# tools — สคริปต์ประกอบงานภาพ

## ทำไมต้องมีโฟลเดอร์นี้

**AI เจนตัวอักษรไทยเพี้ยนเสมอ** — Codex เขียนโลโก้มาเป็น "เป่า ยั้ง ฉุบ!" (ไม้หันอากาศ)
ทั้งที่ต้องเป็น "เป่า ยิ้ง ฉุบ!" (สระอิ) · เกมที่ 1 ก็เคยพังแบบนี้ 3 รอบ

**ImageMagick ก็ใช้ไม่ได้** — วางสระ/วรรณยุกต์ไทยผิด (ไม้เอกใน "เป่า" หายไปเลย)
เพราะไม่มี text shaping ของภาษาไทย

**วิธีที่ใช้ได้จริง: เรนเดอร์ด้วยเบราว์เซอร์** (Chromium จัดวางไทยถูก 100%)

## วิธีสร้างโลโก้ใหม่

1. แก้ข้อความ/สไตล์ใน `logo-text.html`
2. `cd ../app && node ../tools/render-logo-text.mjs` → ได้ PNG พื้นโปร่งใส
3. วางทับกรอบเปล่า `theme-samples/logo-frame-clean.png` ด้วย ImageMagick
4. ผลลัพธ์ไปที่ `app/public/assets/logo.png`

> ต้องรันจากใน `app/` เพราะ playwright ติดตั้งไว้ที่นั่น (ESM ไม่อ่าน NODE_PATH)

## ฟอนต์

ใช้ **Mitr-Bold** (`~/Library/Fonts/Mitr-Bold.ttf`) — ฟอนต์ไทยหัวกลมหนา เข้ากับสไตล์การ์ตูน
เครื่องนี้ไม่มี Kanit ที่เกมที่ 1 เคยใช้
