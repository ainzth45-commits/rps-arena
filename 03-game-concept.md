# 03 — Game Concept (เอกสารสำหรับคนทำงานภาพ)

> เอกสารนี้เขียนให้ **Codex อ่านก่อนเจนภาพทุกครั้ง** — เพื่อให้เข้าใจว่าเกมนี้คืออะไร ใครเล่น และทำไมภาพต้องเป็นแบบนี้

---

## 1. เกมนี้คืออะไร

**"เป่า ยิ้ง ฉุบ! อารีน่า!"** — เกมเป่ายิ้งฉุบสะสมคะแนน เล่นบน iPad เครื่องเดียวที่วางไว้กลางออฟฟิศ

ไม่ใช่เกมมือถือทั่วไป แต่เป็น**เกมของทีมขายจริงๆ** ที่บริษัททำขึ้นเพื่อ:
1. **กระตุ้นยอดขาย** — พนักงานต้องใช้ "เหรียญ" ที่ได้จากยอดขายจริงมาแลกสิทธิ์เล่น (เล่น 1 ครั้ง = 3 เหรียญ)
2. **สร้างบรรยากาศสนุกในที่ทำงาน** — ให้คนในทีมมีเรื่องแซวกัน มีเรื่องลุ้นกันทุกวัน

## 2. ⭐ ใครเล่นเกมนี้ (สำคัญที่สุดสำหรับงานภาพ)

**พนักงานทีมขาย ซึ่ง "ส่วนใหญ่เป็นผู้หญิง"**

เพราะงั้น:
- **ตัวละครในเกมส่วนใหญ่ต้องเป็นผู้หญิง** — ผู้หญิงวัยทำงาน แต่งตัวออฟฟิศ
- หน้าตาสดใส เป็นมิตร ดูเข้าถึงง่าย ไม่ใช่นักสู้ดุดันน่ากลัว
- มีผู้ชายปนได้บ้างแต่ไม่ใช่ตัวหลัก
- **ห้ามทำให้ดูเป็นเกมของผู้ชาย** (ไม่เอาแนวนักสู้กล้ามโต ไม่เอาแนวสงคราม ไม่เอาอาวุธจริง)

### อ้างอิงสไตล์ตัวละครจากเกมที่ 1 (ทีมเดียวกัน จักรวาลเดียวกัน)

เปิดดูไฟล์พวกนี้ก่อนวาดตัวละครนะคะ:

```
เกมที่1/app/public/assets/generated/role-normal.webp   ← พนักงานหญิงออฟฟิศ (ต้นแบบหลัก)
เกมที่1/app/public/assets/generated/role-spy.webp      ← ผู้หญิงหน้ากวนๆ
เกมที่1/app/public/assets/generated/role-jester.webp
เกมที่1/app/public/assets/generated/mascot-detective.webp ← แมวส้มมาสคอต (ตัวเดียวกับเกมนี้)
```

สิ่งที่ต้องเก็บมาจากเกมที่ 1:
- **การ์ตูนผู้หญิงออฟฟิศ ตาโตกลม หน้ากลมมน** ดูเป็นมิตร
- **ขอบเส้นดำหนา** รอบตัว
- **สัดส่วนการ์ตูนน่ารัก** (ประมาณ 6 หัว ไม่ยืดยาวแบบการ์ตูนวัยรุ่น)
- ชุดทำงาน: เบลเซอร์ กระโปรง เสื้อเชิ้ต ผ้าพันคอ
- **พื้นหลังโปร่งใส เต็มตัว**
- แสงเงาน้อยแต่มี ไม่แบนสนิท

## 3. อารมณ์ของเกม

**สนุก ลุ้น มันส์ กวนๆ** — แต่ไม่ก้าวร้าว

เกมนี้คือการที่เพื่อนร่วมงานมาแกล้งกัน แซวกัน ท้ากันสนุกๆ ไม่ใช่ศึกชิงความเป็นความตาย

| ต้องการ | ไม่ต้องการ |
|---|---|
| กวนประสาท ยียวน ทะเล้น | โหดร้าย ดุดัน น่ากลัว |
| โมโหแบบการ์ตูน (ไอพุ่งหู เส้นปูด) | โกรธแบบสมจริง เลือด ความรุนแรง |
| ลุ้นระทึกแบบเกมโชว์ | เครียดกดดันแบบเกมแข่ง e-sport |
| สีสดใส จัดจ้าน | โทนมืดหม่น |

## 4. ตัวละครประจำเกม

**คู่ปรับหลักที่ปรากฏทั่วเกม:**

| ตัวละคร | บุคลิก |
|---|---|
| 🐱 **แมวส้มกวนโอ๊ย** | มาสคอตตัวป่วน ยิ้มเยาะมุมปาก ตาหยี เห็นเขี้ยว ท่าทางชิลๆ เหมือนไม่คิดว่าใครจะสู้ได้ **ตัวเดียวกับสารวัตรแมวในเกมที่ 1** |
| 👩‍💼 **พนักงานหญิงหัวร้อน** | สาวออฟฟิศที่โดนแมวกวนจนหงุดหงิด คิ้วขมวด ตาลุก มีเส้นปูดโกรธที่ขมับ ไอพุ่งออกหู — แต่ยังน่ารัก ไม่น่ากลัว |

เรื่องราวคือ **แมวชอบกวน พนักงานหญิงหัวร้อน** = คู่ปรับที่ตลกและมีเรื่องเล่า
ให้คนดูเห็นแล้วรู้ทันทีว่าใครเป็นตัวป่วน ใครเป็นคนโดนแกล้ง

## 5. กลไกหลักที่งานภาพต้องรองรับ

- ผู้เล่นจ่ายเหรียญ → ท้าดวลเพื่อนร่วมทีม → **เป่ายิ้งฉุบตัดสิน** → ได้/เสียคะแนน
- คู่ต่อสู้ไม่ต้องอยู่ตรงนั้น — ระบบออกมูฟแทนตามชุดที่เจ้าตัวตั้งไว้ล่วงหน้า 3 ช่อง
- มีการไต่อันดับ สะสมสตรีคชนะต่อเนื่อง จบซีซั่นแจกรางวัลจริง

**จังหวะที่ต้องลุ้นที่สุดคือตอนเปิดมูฟ** — นับ "เป่า... ยิ้ง... ฉุบ!" แล้วเปิดพร้อมกันสองฝั่ง

## 7. ⛔ กฎเหล็กของงานภาพ

1. **ทุกภาพต้องเจนด้วยเครื่องมือ generate image เท่านั้น** — ห้ามวาดด้วยโค้ด (SVG/canvas/HTML/สคริปต์วาดรูปทรง) เด็ดขาด
   ใช้ ImageMagick ได้เฉพาะแปลงไฟล์/ย่อขนาด/ตัดพื้นหลัง ของภาพที่เจนมาแล้ว
   ถ้าเครื่องมือเจนภาพใช้ไม่ได้ **ให้บอกตรงๆ อย่าเปลี่ยนไปวาดเอง**
2. **ห้ามมีตัวอักษรใดๆ ในภาพ** ทั้งไทยและอังกฤษ (ตัวอักษรไทยเราประกอบเองด้วยฟอนต์จริง)
3. **พื้นหลังโปร่งใส** ทุกชิ้น ยกเว้นที่ระบุว่าต้องทึบ
4. **เซฟเป็น `.webp`**
