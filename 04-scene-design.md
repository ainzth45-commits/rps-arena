# 04 — Scene Design: ทำให้แต่ละหน้ามี "ฉาก" เฉพาะตัว

> เป้าหมาย: เกมดูมีมิติเหมือนเกมจริง — แต่ละหน้ามีบรรยากาศ สี แสง และการเคลื่อนไหวของตัวเอง
> ไม่ใช่ทุกหน้าใช้พื้นหลังอารีน่าเดียวกัน

---

## หลักคิด: "3 ชั้น" ของแต่ละฉาก

แต่ละหน้าประกอบด้วย 3 ชั้น — ปรับได้อิสระต่อหน้า:

1. **ชั้นพื้นหลัง (mood)** — สี/แสง/บรรยากาศ บอกว่า "เรากำลังอยู่ที่ไหน อารมณ์ไหน"
2. **ชั้นเนื้อหา (content)** — panel/ปุ่ม/ข้อมูล
3. **ชั้นเอฟเฟกต์ (juice)** — animation เข้า/ออก · ประกาย · การสั่น · จังหวะลุ้น

**ของที่ทำด้วย CSS ได้เลย** (ไม่ต้องรอ Codex): gradient เฉพาะหน้า · vignette · spotlight · animation · เส้นสปีด · การจัดรูปผู้เล่น
**ของที่ต้องรอ asset**: ตัวละครท่าทางต่างๆ · ฉากผลการดวล · props ประกอบ

---

## ฉากแต่ละหน้า

### 🏠 หน้าแรก (home) — "สังเวียนก่อนเกม"
- **มี:** ฉากอารีน่าเต็มจอ + เม็ดยาสถานะ + dock
- **ทำเพิ่ม:** ไฟสปอตไลต์กวาดช้าๆ · ฝูงชนขยับเบาๆ (ทำด้วย CSS ทีหลัง)

### 🥷 ห้องปรับแต่งมูฟ (moveSet) — "ห้องลับ"
- **mood:** โทน**มืดลง เข้มขึ้น** เหมือนห้องวางแผนลับ — ม่วงเข้ม/น้ำเงินเข้ม แสงสลัว
- **content:** 3 ช่องมูฟเป็น "การ์ดลับ" · ปุ่มยืนยัน = ปิดผนึก
- **juice:** เมื่อยืนยัน → การ์ดพลิกคว่ำ + จอปิดทับ "ความลับถูกผนึก"
- **ทำด้วย CSS ได้** + ใช้ icon-lock (มีแล้ว)

### ⏱️ ห้องเตรียมตัว (movePick 30 วิ) — "จับเวลากดดัน"
- **mood:** โทน**ตึงเครียด** — พอเวลาเหลือน้อย พื้นหลังเต้นเป็นจังหวะแดง/ส้ม (pulse) เร็วขึ้นเรื่อยๆ
- **content:** มือ 3 ท่าใหญ่ · นาฬิกาเด่นกลางบน
- **juice:** ≤10 วิ จอสั่นเบาๆ + ขอบจอเรืองแดง · เลือกมูฟแล้วมูฟเด้งขึ้น
- **ทำด้วย CSS ได้ทั้งหมด**

### ⚔️ หน้า VS — "ปะทะ!" ← ทำก่อน (impact สูงสุด)
- **mood:** ผู้เล่นพุ่งเข้าหากันจากสองข้าง · มุม**น้ำเงิน (ผู้ท้าชิง) vs ชมพู (คู่แข่ง)**
- **content:** **รูปผู้เล่นจริง 2 ฝั่ง** ใหญ่เต็มตัว · ชื่อ+อันดับ · ตรา VS + ฟ้าผ่ากลาง
- **juice:** สองฝั่งเลื่อนพุ่งเข้าชนกลางจอ + ฟ้าผ่าแตก + จอสั่น 1 ที
- **template:** ออกแบบเป็นกรอบ ใส่รูปผู้เล่นคนไหนก็ได้ (เจ้านายเสนอถูก)
- **ทำด้วย CSS + รูปผู้เล่น ได้เลย** · เสริมด้วย clash-spark (รอ Codex)

### 👊 ฉากเป่ายิ้งฉุบ (shoot) — "ลุ้นสุด!" ← ทำก่อน
- **mood:** เวทีมืดลง สปอตไลต์ 2 ดวงส่องผู้เล่นสองฝั่ง
- **content:** **รูปผู้เล่น 2 ฝั่ง** + มือของแต่ละฝั่งชูขึ้น
- **juice:** นับ "เป่า...ยิ้ง...ฉุบ!" ตัวใหญ่เด้งกลางจอ → เปิดมือพร้อมกัน + ประกายปะทะ + จอกระพริบ
- **ทำด้วย CSS + รูปผู้เล่น + มือ (มีแล้ว)**

### 🏆 ฉากผลการดวล (duelResult) — "ตัดสิน" ← ต่างกันตามผล
- **ชนะ:** พื้นทอง ประกายฉลอง confetti · รูปผู้เล่นเด้งดีใจ
- **แพ้:** พื้นเทาหม่น ฝนตก · รูปหงอย
- **เสมอ:** พื้นครึ่งน้ำเงินครึ่งชมพู ประกายชนพอดี
- **ทำด้วย CSS ได้ + เสริม result-win/lose/draw (รอ Codex)**

### 📬 จอ "ระหว่างที่คุณไม่อยู่" — "เปิดกล่องจดหมาย"
- ซองจดหมายเปิด → รายการเลื่อนออกมาทีละอัน · ป้ายเตือนสีแดงกระพริบถ้าโดนไล่เก็บ

### 🏅 ประกาศแชมป์ (seasonEnd) — มีโพเดียมแล้ว
- **ทำเพิ่ม:** สปอตไลต์ส่องแชมป์ · confetti ทอง · ฝูงชนโห่
