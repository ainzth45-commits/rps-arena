## 1. ทิศทางงานภาพ

**การ์ตูนแบนกวนๆ แบบเกมมือถือ casual**

| องค์ประกอบ | ข้อกำหนด |
|---|---|
| ลายเส้น | **ขอบดำหนาชัด** รอบทุกชิ้น |
| สี | **สีแบน สดจัด** ไม่ไล่เฉดเยอะ ไม่มีพื้นผิวสมจริง |
| อารมณ์ | ตลกเกินจริง กวนประสาท ตาโต ปากกว้าง |
| รูปทรง | เรียบง่าย อ่านออกทันทีแม้ย่อเล็ก |
| เอฟเฟกต์ | ประกายดาว เส้นสปีด รัศมีพุ่ง ฟ้าผ่า |

**โทนสี (ตรงกับ CSS ที่เขียนไว้แล้วใน `app/src/styles/global.css`)**

| ตัวแปร | ค่า | ใช้กับ |
|---|---|---|
| `--ring-blue-deep` | `#16226b` | พื้นหลังฝั่งน้ำเงิน |
| `--ring-blue` | `#2b4bd8` | มุมน้ำเงิน (ผู้เล่น) |
| `--ring-purple` | `#6a2fd0` | พื้นหลังกลาง |
| `--ring-magenta` | `#e82f9c` | มุมชมพู (ผู้ท้าชิง) |
| `--spark` | `#ffd93d` | ปุ่มหลัก · ตัวเลขคะแนน · เน้น |
| `--ink` | `#10121f` | ขอบดำทุกชิ้น |
| `--paper` | `#fdf7ff` | ตัวหนังสือ |

**ตัวละครประจำเกม**
- 🐱 **แมวส้มกวนโอ๊ย** — ต่อยอดจากมาสคอตเกมที่ 1 (สารวัตรแมว) เป็นตัวเดียวกันในจักรวาลเดียวกัน
- 👩‍💼 **พนักงานหญิงหน้าโมโห** — โดนแมวกวนแล้วหัวร้อน มีเส้นปูดโกรธ + ไอพุ่งออกหู

## 3. กฎเรื่องพื้นหลัง (บทเรียนจากเกมที่ 1 — พลาดมาแล้ว)

| ประเภท | พื้นหลัง |
|---|---|
| ไอคอน · ตัวละคร · โลโก้ · เอฟเฟกต์ | **โปร่งใส (มี alpha)** |
| ฉากจบ · พื้นหลังเกม · **ไอคอนแอป** | **ทึบเต็มเฟรม (ห้ามมี alpha)** |

> ⚠️ **ไอคอนแอป iOS ห้ามมี alpha เด็ดขาด** — ถ้ามี iOS จะเรนเดอร์พื้นดำ

## 4. ⛔ ห้ามให้ AI เจนตัวอักษรไทย

โมเดลเขียนภาษาไทยเพี้ยนเกือบทุกครั้ง — เกมที่ 1 เคยพัง 3 รอบติดกับป้าย "พลาด!"

**วิธีที่ใช้จริงและได้ผล:**
1. ให้ Codex ทำ **"แผ่นป้าย/กรอบเปล่า"** ไม่มีตัวอักษร
2. ฟรายเดย์ประกอบตัวอักษรเองด้วย **ImageMagick + ฟอนต์ Kanit ตัวหนา**

เพราะงั้นโลโก้ (A1) จึงสั่ง Codex ไว้ **2 เวอร์ชัน**: มีตัวอักษร กับ กรอบเปล่า — ถ้าเวอร์ชันมีตัวอักษรสะกดเพี้ยนแม้แต่นิดเดียว ใช้กรอบเปล่าแล้วประกอบเองทันที ไม่ต้องเจนซ้ำ
